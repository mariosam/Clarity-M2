********************************************************************************
*** MICROSOFT CLARITY MODULE - READ ME (english)
********************************************************************************

*** Objectives

The objective of this module is to insert in all pages of your theme the
analitycs code of Microsoft Clarity.

*** Version 1.0.0

What we have in this version:
- inform the tracking ID and insert the code in all pages of theme.

*** Tested

Version: 2.4.1 Open

PHP: 7.3.9

Theme: LUMA

*** Install

Move the <Clarity> folder to the directory:
app/code/MarioSAM/ (create the directory if it doesn't exist)

At the command prompt, Magento installation directory, type:
$ bin/magento module:enable MarioSAM_Clarity
$ bin/magento setup:upgrade
$ bin/magento cache:clean

Access the Magento panel and navigate to:
Stores > [Settings] Configuration > Mario SAM > clarity Settings
